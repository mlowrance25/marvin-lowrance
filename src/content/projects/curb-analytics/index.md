Designed and built at INRIX. An interactive map application that surfaces parking rules and metrics for any street in the world — combining static street data with dynamic predictive parking and traffic analytics.

Backed by APIs handling 4M+ events per hour, ingested via Kafka, Kinesis Firehose, and NATS.io, snapped to curb geometry and served at P98 latency under 1 second.

**Stack:** React, Next.js, TypeScript, Python, AWS, Kafka, NATS.io, MapBox
